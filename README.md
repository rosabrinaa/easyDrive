# easyDrive
Projeto acadêmico voltado ao planejamento de um sistema web para gestão de locação de veículos.

Principais atividades:

• Desenvolvimento do protótipo da interface no Figma
• Estruturação do fluxo de navegação do sistema
• Definição das funcionalidades principais da aplicação
• Planejamento do projeto utilizando metodologia Scrum
• Organização das atividades em sprints

Competências desenvolvidas:

Prototipação • UX/UI • Planejamento de produto • Metodologias ágeis.
